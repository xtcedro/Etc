Declaración de Accesibilidad

Fecha de Vigencia: 1 de enero de 2024

1. Compromiso con la Accesibilidad

En Dominguez Tech Solutions, estamos comprometidos a proporcionar un sitio web accesible para todos, independientemente de sus habilidades o discapacidades. Nuestro objetivo es garantizar que todos los usuarios tengan una experiencia fluida al navegar por nuestro sitio web y acceder a nuestros servicios.

2. Estándares de Accesibilidad

Este sitio web ha sido diseñado para cumplir con las Pautas de Accesibilidad al Contenido en la Web (WCAG) 2.1, Nivel AA. Estas pautas ayudan a garantizar que el contenido sea accesible para una gama más amplia de personas con discapacidades, incluidas aquellas con ceguera, baja visión, pérdida de audición y discapacidades cognitivas.

3. Funciones de Accesibilidad de Nuestro Sitio Web

  • Alternativas de texto para contenido no textual.
  • Navegación por teclado para todos los elementos interactivos.
  • Relaciones de contraste que hacen que el texto y los elementos visuales sean legibles.
  • Diseño adaptable que se ajusta a diferentes tamaños de pantalla y tecnologías de asistencia.

4. Mejoras Continuas

Continuamos trabajando para mejorar la accesibilidad de nuestro sitio web. Los comentarios de nuestros usuarios son fundamentales para ayudarnos a lograr nuestros objetivos.
