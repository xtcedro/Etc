Declaración de Accesibilidad

Nuestro compromiso de garantizar que nuestro sitio web sea accesible para todos.

Fecha de Vigencia: 1 de Enero de 2025

1. Nuestro Compromiso con la Accesibilidad

En Dominguez Tech Solutions, estamos comprometidos a proporcionar un sitio web accesible para todos, independientemente de sus habilidades o discapacidades. Nuestro objetivo es garantizar que todos los usuarios tengan una experiencia fluida mientras navegan por nuestro sitio web y acceden a nuestros servicios.

2. Estándares de Accesibilidad

Este sitio web ha sido diseñado para cumplir con las Pautas de Accesibilidad para el Contenido Web (WCAG) 2.1, Nivel AA. Estas pautas ayudan a garantizar que el contenido sea accesible para una gama más amplia de personas con discapacidades, incluidas aquellas con ceguera, baja visión, pérdida de audición y discapacidades cognitivas.

3. Características de Nuestro Sitio Web Accesible

  • Alternativas de texto para contenido no textual.
  • Navegación con teclado para todos los elementos interactivos.
  • Relaciones de contraste que hacen que el texto y los elementos visuales sean legibles.
  • Diseño responsivo que se adapta a diferentes tamaños de pantalla y tecnologías asistivas.

4. Mejoras Continuas

Estamos trabajando continuamente para mejorar la accesibilidad de nuestro sitio web. Los comentarios de nuestros usuarios son vitales para ayudarnos a alcanzar nuestros objetivos.
